# Todo_Bubble
투두리스트를 멋쟁이사자처럼 프론트엔드스쿨 3기 동료와 둘이서 만들어봤습니다.

React, Styled-components를 사용했습니다.

각 리스트들은 배경에서 위로 올라가며, 한번 클릭시 컬러가 바뀌고. (체크를 의미합니다.) 더블클릭시 삭제가 됩니다.
